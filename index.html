What I've Learned This Year

This year I've learned a lot about computers. In the image manipulation unit the main thing I learned was how to use GIMP. I learned how to remove the alpha channel and how tpo mix images together. In the HTML and CSS unit I learned how to code. I learned all of the simple features of these two languages, and I learned some complex features. I also learned how to use some bootstrap and some javascript. In the 3D modeling unit I learned how to use Blender and Unreal 4. In Blender we used tutorials to make complex models, and we used the information we learned to make our own models. In Unreal 4 we learned some of the simple features in order to make our biosphere. In the CS unplugged unit I learned all about how computers work. Even though this was through simple lessons, I still got a good understanding. That was a summary of the things I've learned this year.
